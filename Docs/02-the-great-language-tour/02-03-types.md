# Types

## Overview

### Primitives

- `int` for integers
  - `0`, `1`
- `float` for floating-point numbers
  - `3.14`, `.1`, `2.`
- `str` for strings
  - `'string'`, `"string"`
- `bool` for Boolean values
  - `True`, `False`
- `None` for Null / no value

### Collections

- `list` for lists (ordered)
  - `["string", 1]`
- `set` for sets (unordered)
  - `{0, "blah"}`
- `hash` for hash maps (key-value pairs)
  - `%{"name" : "Li", "ability" : "exists"}`
